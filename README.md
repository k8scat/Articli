**Articli** 通过解析 `Markdown` 文件内容以及调用不同平台的接口，实现内容快速在不同平台进行发布。

## 支持的平台

### 平台文章管理

- [掘金](https://juejin.cn)
- [CSDN](https://csdn.net)
